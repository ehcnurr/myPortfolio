Personal/Academic Projects

Umbrellock

Umbrellock is an innovative umbrella storage solution developed using Arduino. It features a locking mechanism with Servo Motors that secure umbrellas in place.

Umbrella ownership is tracked through RFID scanners and tags, ensuring that only authorized users can access each container.

Floodsory

Floodsory is a flood monitoring system designed to provide real-time data on water levels in flood-prone areas. It utilizes Arduino and various sensors to monitor water levels and send alerts when thresholds are exceeded.

The system is designed to help communities prepare for and respond to flooding events, enhancing safety and preparedness.

TaxCalc

TaxCalc is a simple Tkinter application that allows users to calculate their income tax based on their income and applicable tax rates. It provides a user-friendly interface for quick calculations.
